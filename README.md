# Start-Clicking
Auto Clicker Created for a Roblox Game Needs Max Clicks

This Program came into been for a Roblox Running Game. More Clicks Quicker Speed has Key Listner f1 to start f2 to Stop GUI is there Start clicking zip folder contains exe file

1) install python 
2) from powershell cd into folder with source code
3) pip install -r requirements
4) pip install pyinstaller
5) pyinstaller --onefile --windowed autoClicker.py hit enter look for dist folder created move icons folder into dist folder rename as you like

or follow steps 1, 3 and run exe file
